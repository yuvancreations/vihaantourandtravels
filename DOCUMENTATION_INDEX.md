## üÜò Need Help?

### Common Issues

**Q: Links not working?**
```
Check QUICK_REFERENCE.md > Page File Names
Ensure file names match exactly
```

**Q: Header/footer not showing?**
```
Verify CSS and JS files are linked
Check paths are correct in <head> and before </body>
```

**Q: Mobile menu not working?**
```
Ensure Bootstrap 5.3.0 is loaded
Check Script.js is loaded
Test with different browser
```

**Q: Colors not changing?**
```
Edit style.css :root variables
Clear browser cache (Ctrl+Shift+Delete)
Reload page
```

**Q: Forms not submitting?**
```
Check if backend is needed
Add form submission handler to Script.js
Test in browser console
```
